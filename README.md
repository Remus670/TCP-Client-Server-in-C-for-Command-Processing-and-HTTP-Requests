Acest proiect implementează o aplicație client-server folosind socket-uri TCP în limbajul C. Serverul ascultă conexiuni de la clienți și procesează comenzile primite. Pentru o comandă specifică (14), serverul inițiază o cerere HTTP către un alt server și salvează răspunsul într-un fișier. Clientul trimite comenzi și primește răspunsuri corespunzătoare de la server. Programul evidențiază utilizarea funcțiilor de rețea precum socket, bind, listen, accept, connect, send și recv.
