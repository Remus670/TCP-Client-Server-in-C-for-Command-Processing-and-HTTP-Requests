This project is a laboratory assignment completed in 2025 and implements a client-server application using TCP sockets in the C programming language. The server listens for incoming client connections and processes the received commands. For a specific command (14), the server sends an HTTP request to another server and stores the response in a file. The client is responsible for sending commands and receiving the corresponding responses from the server. The project highlights the use of essential networking functions such as socket, bind, listen, accept, connect, send, and recv
